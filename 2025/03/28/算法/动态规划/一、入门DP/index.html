1.2 打家劫舍

题型特点

这些题目通常有一个相邻元素不能同时选取的限制。例如，不能抢劫相邻的两栋房子或在一个特定的布局中不能选择相邻的两个元素。

解题方法

选或者不选的思路，分两种情况判断。

几个变种

740. 删除并获得点数

100316. 施咒的最大总伤害

板子

这两道题不是相邻元素不能选取，它是给你一个区间，区间范围内的元素不能选取

我们可以先排序，然后从后往前选取，这样只需要考虑前面的，不需要考虑后面的。

我们用 i 代表下标，i 是从0 到 n - 1，所以总共有n个状态

边界条件是i < 0

然后我们定义一个j = i - 1,利用循环找到第一个不在区间内的下标，作为下一次递归传入的对象。

说到底，还是只需要考虑两个变量，一个是当前的，一个是第一个不在区间范围内的，用max(dfs不选，选 + dfs第一个不在区间内的)
